SupaFill

Fill everything in a snap.

A browser extension that fills forms with realistic fake data. Smart detection, consistent personas, 10 locales. One click.

Add to Chrome Add to Firefox

Features

Smart Form Detection

Automatically detects input types: names, emails, phones, addresses, and more.

Consistent Personas

Every form fill uses a coherent identity: matching name, email, and username.

Multi-Locale Support

10 locales including English, German, French, Spanish, Japanese, Korean, and more.

Flexible Filling

Fill entire forms, single fields, or specific sections via right-click menu.

Keyboard Shortcuts

Ctrl+Shift+E to fill, Ctrl+Shift+L to clear. Customizable in settings.

Customization

Add custom field rules, ignore specific fields, set fixed values.

Privacy First

100% local processing. No tracking, no analytics, no account required.

Lightweight

~100KB total. No runtime dependencies. No external API calls.

How It Works

1

Install

Add SupaFill from your browser's extension store. No sign-up needed.

2

Open Any Form

Navigate to any website with a form you need to fill.

3

Fill in One Click

Click the extension icon or press Ctrl+Shift+E. Done.
